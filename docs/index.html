SCLS

What is SCLS ?

SCLS is a C++ workspace, made to simplify the use of the C++ language to the most of people possible.

The SCLS libraries ?

Each SCLS libraries are described here.

SCLS Foundation "Leonhard"

SCLS Foundation "Leonhard" is the foundation of the libraries. It is made to store some very low-level features, like binary manipulation.

You can access to the documentation of this library, here.

Use SCLS with SCLS Workspace

Using SCLS alone, it's good. However, use it in a place made for it, this is better. SCLS Workspace is a workspace made to simplify the use of SCLS to everyone which wants to use SCLS.
Access to the page of SCLS Workspace here.
