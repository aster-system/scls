Presentation

What is SCLS ?

SCLS, for "Simple C++ Libraries Set", is a sets of C++ library, which means to make C++ development almost as easy as in Python. The main goal of the library is to be simply used, implemented, downloaded and reviewed.

Content

The foundation of SCLS

The foundation of SCLS are cutted in 3 mains library:

  • SCLS Foundation "Leonhard", the root of SCLS.
  • SCLS Math "Carl", use advanced maths with SCLS.
  • SCLS Image "Michelangelo", handle images with SCLS.

Make software with SCLS

Even if the realisation of software is way more complicated than what we already saw, you can still do it easily with SCLS:

  • SCLS Graphic "Benoit", use a confortable GUI in C++.
