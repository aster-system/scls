Displaying things in a computer is a process more complicated than it looks. Fortunately, SCLS Graphic "Benoit" can do it easily for you.

Presentation

What is SCLS Graphic "Benoit" ?

SCLS Graphic "Benoit" is one of the easiest way to make your software dream come to reality. It is named after the mathematician Benoit Mandelbrot, which discovered the very beautiful fractal of Mandelbrot. This library uses a lot of intuitive ways to make a GUI, with a lot of concepts, like an XML text-to-GUI formater (similar to HTML).
